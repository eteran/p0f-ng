Installation guide:
  Debian 10:
    You need this packages installed at your system:
      root@pof:/home/pof/# apt-get install cmake build-essential python-dev libpcap-dev libboost-all-dev

      Let's clone actual repo:
          root@pof:/home/pof/# git clone https://github.com/eteran/p0f-ng.git
          root@pof:/home/pof/# cd p0f-ng/

      If you need only p0f package:
          root@pof:/home/pof/# cmake .

      If you need tools:
          root@pof:/home/pof/# cmake -D P0F_BUILD_TOOLS=ON .

      And finally run make
          root@pof:/home/pof/# make
          
  CentOS 7:
    You need this packages installed at your system:
      [root@localhost ~]# yum install cmake3 cpp gcc boost169-devel centos-release-scl devtoolset-8 libpcap-devel -y
      
    Let's clone actual repo:
        root@pof:~# git clone https://github.com/eteran/p0f-ng.git
        root@pof:~# cd p0f-ng/
        
    Activate gcc version 8:
      [root@localhost p0f-ng]# scl enable devtoolset-8 bash
      
    If you need only p0f package:
      [root@localhost p0f-ng]# cmake3 . -D BOOST_INCLUDEDIR=/usr/include/boost169/
    
    If you need tools:
      [root@localhost p0f-ng]# cmake3 . -D BOOST_INCLUDEDIR=/usr/include/boost169/ -D P0F_BUILD_TOOLS=ON
      
    And finally run make
      [root@localhost p0f-ng]# make
